리눅스 자격증 통합 준비 과정의 50p 문제
계정 설정의 연습문제
사용자 www-hacker, www-user, data-user 그리고 db-user를 생성한다.
모든 사용자는 비밀번호를 rockylinuxworld라고 설정한다.
사용자 www-hacker는 웹 서비스에 접근은 가능하나, 시스템에 접근은 불가능하다.
www-user라는 그룹에 포함이 된다.
www-user,data-user는 secure-data그룹에 포함이 된다.
각 사용자는 tcsh를 통해서 시스템에 접근이 가능하다.
db-user는 wheel그룹에 포함이 되며, 시스템에 로그인이 가능하다.
각각 사용자를 ssh로 접근 후 "id"명령어 실행한다.

chgrp : 그룹 권한만 지원
chgrp를 사용하는 사용자가 해당 파일의 소유권을 가지고 있어야 하고, 변경하려는 그룹에 그룹원으로 가입을 하고 있어야 한다.

-rw-r--r--. 1 root root 0 Jun 12 14:21 file1

rw-r--r-- -> 권한의 표시
rwxrwxrwx > 반복 되어 있다.
rwx|rwx|rwx
[U] [G] [O]
U user : 사용자
G group : 그룹사용자
O other : 전체 - {사용자, 그룹사용자}

모드 변경의 두가지 방법
심볼릭 모드 : 문자를 통하여 변경하는 방법 < 권장 
옥탈 모드 : 숫자를 통하여 변경하는 방법

chmod 777   사용은 가능하지만 비권장 8진수로 하는 경우 숫자 계산 필요
chmod a=rwx GNU에서는 이 방식을 매우 권장

rw-r--r--
--- --- ---
 U   G   O
심볼릭 모드 이해를 위한 문자
u User 소유자의 권한
g Group 그룹소유자의 권한
o Other 소유자,그룹소유자를 제외한 전부
a All U+G+O의 권한

+ 권한을 추가
- 권한을 제거
= 기존 권한을 무시하고 새로운 권한으로 업데이트 

r read 읽기
w write 수정
x execute 실행
X 디렉토리에만 적용되는 권한 
: 디렉토리는 x권한을 부여하고 파일의 경우에는 기존에 x권한이 있다면 x권한을 부여하고 없다면 x권한을 부여하지 않는다.

파일과 디렉토리의 권한 부여 차이
FILE
R 파일을 읽음 * 
W 파일을 수정
X 파일을 실행
--x -wx -w-  [r-- rw- rwx r-x ]

DIR : 명령어에 대한 권한
R 디렉토리 읽음 : ls 명령어
W 디렉토리를 수정 : cp mv rm 명령어
X 디렉토리를 실행 : cd 명령어* 

[root@node1 test]# chmod u=rw- file1
[root@node1 test]# chmod g+rx file1
[root@node1 test]# chmod o=rwx file1
[root@node1 test]# chmod u+x file1
[root@node1 test]# chmod g-x,o-x file1
[root@node1 test]# chmod go-x file1
[root@node1 test]# chmod a=rw file1
