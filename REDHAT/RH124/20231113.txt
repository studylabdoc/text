명령어 사용의 문법

명령어 3대 블록
# [1][2][3]
옵션과 인자 명령어 사이는 항상 공백 문자를 추가
기준값 공백 -> 토큰 
[1] 명령어 : 항상 명령어는 쉘의 처음에 온다.
보통 일반 문자열로 되어 있다.

[2] 옵션 : 명령어의 형식을 변경
보통 -나 --로 되어 있으며 문자열 형식의 옵션도 존재
-와 --가 같은 뜻을 가진 명령도 있고
또는 -만 있는 명령도 있다.
# ls --all
# ls -a
# ps aux
옵션의 순서가 보통은 상관이 없고 섞어 쓰기 가능
# ls -ia
#  ls -ai
#  ls -a -i
#  ls --all -i
[3] 인자 : 위치를 변경하지 않고 명령어의 대상을 지정
주의점: 옵션에 인자가 결합되어 있는 경우에는 옵션과 인자의 순서에 주의한다.
