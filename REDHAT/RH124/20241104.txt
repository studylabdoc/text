mv
inode 번호를 변경하지 않고 파일의 이름만 변경 한다.
같은 파일 시스템 내에서만 inode 번호가 변경되지 않는다.
다른 파일 시스템으로 이동을 한다면 파일을 복사하고 원본 파일은 삭제.
# mv file1 file2 file3 dir
여러개의 파일을 이동 할 때 마지막에는 디렉토리를 작성
# mv file1 file2
동일한 아이노드를 가지고 있지만 파일 이름만 변경
# mv file1 dir1
dir1으로 file1 이름을 유지한채 이동
# mv file1 dir1/file10
dir1으로 이름을 file10번으로 변경하며 이동
